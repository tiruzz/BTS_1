Riposte:

1) Semplificare lo sviluppo web attraverso un framework front-end con componenti predefiniti.

3) Il foglio di stile CSS (bootstrap.min.css) e, se necessario, il file JavaScript (bootstrap.min.js) da un CDN.

3) Una rete di server che fornisce file statici. Vantaggi: velocità di caricamento, cache, affidabilità.

4) Definire la codifica dei caratteri del documento, ad esempio, UTF-8.

5) "Container" è a larghezza fissa centrata, mentre "Container Fluid" occupa l'intera larghezza della finestra del browser.
